Call Centre Dashboard

A business-focused dashboard built with HTML, CSS, and JavaScript for tracking agent attendance, team performance, sales activity, and internal reporting in one responsive management view.

GitHub Live Demo

GitHub Live Demo

Ebook Reader App

A responsive ebook reader demo built with HTML, CSS, and JavaScript, featuring a dynamic bookcase interface and mobile-first reading experience simulation.

The project demonstrates front-end development, UI/UX design thinking, and scalable product planning for future mobile app deployment.
